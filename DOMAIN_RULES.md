# Domain Rules (skrót lokalny)

## Niezmienne reguły

- Nie zmieniaj po cichu bilansowania faz.
- Nie zmieniaj po cichu progów walidacji.
- Nie psuj relacji RCD -> MCB/RCBO.
- Nie zmieniaj semantyki danych trafiających do PDF.

## Tryb pracy przy ryzyku

Jeśli zachowanie wygląda na błędne:

1. Opisz obecny stan.
2. Opisz ryzyko i wpływ.
3. Zaproponuj minimalną poprawkę.
4. Zabezpiecz zmianę testem.
