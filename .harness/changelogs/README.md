# Changelog convention

This directory tracks per-day commit changelogs for DINBoard Web, per the project convention (see `AGENTS.md` and `README.md`).

Add files named `YYYY-MM-DD.md` with a short, dated summary of commits that day. Keep entries concise — a few lines per change. Reference the commit SHA and the subsystem (electrical / canvas / pdf / project-io / ui / tests / docs).

Skeleton:

```markdown
# YYYY-MM-DD

## <subsystem>
- <one-line description of change> (<short-sha>, by <author>)

## <subsystem>
- ...
```
